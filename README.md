# ShopBanHang

Ứng dụng quản lý bán hàng giúp quản lý khách hàng và sản phẩm một cách đơn giản và hiệu quả.

---

## 🚀 Chức năng chính

- Sắp xếp sản phẩm theo giá (tăng dần / giảm dần)
- Lọc sản phẩm theo điều kiện
- Quản lý khách hàng:
  - Thêm khách hàng
  - Sửa thông tin
  - Xóa khách hàng
  - Cập nhật dữ liệu

---

## 🛠 Công nghệ sử dụng

- HTML / CSS / JavaScript(React)
- Python(FastAPI)
- Git & GitHub

---

## 📌 Ghi chú

Dự án phục vụ mục đích học tập và thực hành kỹ năng lập trình.
